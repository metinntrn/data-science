This project consists of two main stages:

    Data Cleaning: Handling missing data, correcting erroneous entries (e.g., cleaning outliers in the attendance column), and preparing categorical data for analysis.

    Data Visualization: Creating interactive charts using Plotly and examining correlations (relationships) between variables.

📊 Key Findings

The following key insights emerged from the analysis:

    Attendance Rate: There is a strong positive correlation between attendance rates and final grades. Consistent attendance is one of the most critical factors driving academic success.

    Study Hours (StudyHoursPerWeek): A significant increase in final grades was observed as the number of weekly study hours increased.

    Online Education: It was observed that participation in online classes does not have a sole determining effect on final grades.

        Insight: "Box plot analysis reveals no significant correlation between the number of online classes taken and final grades. This implies that online classes alone do not drive performance, suggesting that other factors—most notably StudyHoursPerWeek—have a greater impact on student success."
