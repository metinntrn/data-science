# 🤖 LLM Usage Dashboard
**app link**:https://9ofhmnmp9szlys4pykdaru.streamlit.app/
An interactive Streamlit dashboard for analyzing Large Language Model (LLM) usage data.

## 📊 Features

- Filter data by **Model Name**
- Filter data by **Application Domain**
- View key performance metrics
- Interactive Plotly visualizations
- Explore the complete dataset

## Dashboard Metrics

- Average User Satisfaction
- Average Latency
- Total Requests

## 📈 Visualizations

- Average Success Rate by Model
- Average User Satisfaction by Model
- Usage by Application Domain
- Token Count vs Estimated Cost
- Average Latency by Model
- RAG Enabled vs Success Rate

## 🛠️ Technologies

- Python
- Streamlit
- Pandas
- Plotly Express

## 📂 Dataset

The dashboard uses the following dataset:

```
genai_llm_usage_dataset_1000.csv
```

The dataset includes information such as:

- Model Name
- Application Domain
- User Satisfaction
- Successful Response
- Latency
- Token Usage
- Estimated Cost
- RAG Enabled
